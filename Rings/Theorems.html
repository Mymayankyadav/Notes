2.2.5 Theorem

If \(R\) is a ring with 1 and \(I\) is a ideal in \(R\) such that \(I \neq R\), then there is a maximal ideal \(M\) of the same kind as \(I\) such that \(I \subseteq M\)

    Remarks:
  1. The above theorem is not true if R has no unity.
  2. Above theorem need not be true for minimal ideals of R even if R is commutative with 1.

2 .2.9 Theorem

The set of all nilpotent elements in a commutative ring \(R\) with 1 is the intersection of all prime ideal, i. e., \(N = N'\).

2.4.1 Proposition

Suppose \(R\) is a ring with 1. Let I be a left ideal. Then the following are equivalent.

  1. \(I\) = \(R\).
  2. 1 \(\in\) \(I\).
  3. \(I\) contains a unit.
  4. \(I\) contains an element which has a left inverse.

2.4.5 Proposition

Suppose \(R\) contains 1, and \(I\) is a 2-sided ideal of R. Then I = R \(\iff\) 1 \(\in\) \(I\) \(\iff\) I contains a unit \(\iff\) I contains an element which has a left inverse or a right inverse.

2.4.6 Theorem

Let \(R\) be a ring with 1 . Then \(R\) is a division ring \(\iff\) (0) and \(R\) are the only left ideals in \(R\) \(\iff\) (0) and \(R\) are the only right ideals in \(R\).

2.4.9 Theorem

Let \(R\) be a non-zero ring in which the multiplication is non-trivial. Then \(R\) is a division ring if and only if \(R\) has no left ideals other than (0) and \(R\), ( i. e., \(R\) contains a pair x , y such that xy\(\neq\)0. Then if \(R\) has no left (resp. right) ideals other than (0) and \(R\), \(R\) has 1 and \(R\) is a division ring) .

2.4.10 Theorem

Let R be a ring with 1 and \(S = M_n(I)\) , n \(\geq\) 1. Then we have the following.

  1. For a left ideal \(J\) in \(R\), \(M_n(I)\) is a left ideal in \(S\).
  2. For a right ideal \(I'\) in \(R\), \(M_n(I)\) is a right ideal in \(S\).
  3. For a 2-sided ideal \(J\) of \(R\), \(M_n(I)\) is a 2-sided ideal of \(S\).
The converse is true only for 2-sided ideal, i. e., every two sided ideal \(K\) in \(S\) is of the form \(K = M_n(I)\)) for some 2-sided ideal \(J\) in \(R\).

2.5.2 Proposition

  1. Sum of two ideals of the same kind is again an ideal of the same kind.
  2. Addition of ideals is commutative and associative.

2.5.4 Proposition

  1. Product of two ideals of the same kind is again an ideal of the same kind.
  2. Product of ideals is associative but need not be commutative.
    Remarks:
  1. \(IJ\) is a left ideal if \(I\) is a left ideal and \(J\) any subset of \(R\).
  2. \(IJ\) is a right ideal if \(I\) is any subset of \(R\) and \(J\) is a right ideal. Furthermore, if \(R\) has 1, then \(IR = I = RI\). However, if \(R\) is without 1 , then \(IR \subseteq I\) and \(RI \subseteq I\) (equality need not hold).
  3. The set of all additive subgroups of \( R, + \) is a commutative a emigroup with identity under addition but not a group.
  4. The set of all left/right/2-sided ideals is a sub-semigroup with identity under addition but not a group.
  5. The set of all left/right/2-sided ideals in \(R\) is a semigroup with identity (namely, \(R\)), under multiplication (provided \(R\) has 1 ) but not a group.

2.5.11 Proposition

Let \(R\) be a commutative ring. Then a nil ideal \(I\) is nilpotent if \(I\) is finitely generated.

2.6.2 Theorem

The natural multiplication of cosets of \(I\), namely, \((a + I) (b + I) = ab + I\), make \(R/I\) into a ring if and only if \(I\) is a 2-sided ideal. For a 2-sided ideal \(I\), \(R/I\) is called the quotient ring or residue ring of \(R\) modulo \(I\).

2.7.1 Theorem

Let \(I\) be a 2-sided ideal in \(R\) and \(S = R/I\), the quotient ring of \(R\) modulo \(I\). Then we have the following.

  1. The set of all subgroups of \((R/I, +)\) is naturally bijective with the set of all subgroups of \(R\) each containing \(I\).
  2. The set of all subrings of \(R/I\) is naturally bijective with the set of subrings of R each containing \(I\).
  3. The set of all left/right/2-sided ideals of \(R/I\) is naturally bijective with the set of left/right/2-sided ideals of \(R\) each containing \(I\).

2.7.2 Theorem

Let \(R\) be a commutative ring with 1 and \(I\) be an ideal in \(R\). Then we have the following.

  1. \(R/I\) is an integral domain if and only if \(I\) is a prime ideal in \(R\).
  2. \(R/I\) is a field if and only if \(I\) is a maximal ideal in \(R\). If \(I\) is maximal, \(R/I\) is called the residue field of \(R\) at \(I\).
  3. A maximal ideal ( in a commutative ring) is a prime ideal but not conversely.

2.7.4 Corollary

For \(2 \leq n\in \mathbb{N}\), the ring \(\mathbb{Z}/n \mathbb{Z} \) is a field \(\iff\) \(\mathbb{Z}/n\mathbb{Z}\) is an integral domain \(\iff n\) is a prime number.

2.7.5 Proposition

For n \(\geq\) 2, the ring \(\mathbb{Z}/n\mathbb{Z} \) has no non-trivial nilpotent elementa if and only if \(n\) is square free.

2.7.6 Theorem

Suppoae \(R\) ia a commutative ring with 1 . Then \(a(X) = a_0 + a_1X + a_2X^2 + · · · + a_TX^T \in R[X]\) is a unit in \(R[X]\) if and only if \(a_0\) is a unit in R and \(a_1, a_2, · · · , a_T\) are all nilpotent in \(R\).

2.8.1 Proposition

Let \(R\) be a ring with 1 . Then the set \(M_R\) of all non-units in \(R\) is precisely the union of all proper left/right ideals of \(R\) which in turn is also equal to the union of all maximal left/right ideals of \(R\).

2.8.3 Proposition

A ring \(R\) with 1 is a local ring if and only if it has a unique maximal ideal, maxima/ simultaneously as a left/right/2-sided ideal.

2.8.8 Proposition

For n\(\geq\) 2, \(\mathbb{Z}/n\mathbb{Z} \) is a local ring if and only if \(n\) is a power of a prime.

Chapter 3

3.1.5 Theorem

Let \(I\) be an ideal in a ring \(R\). Then \(I\) is a 2-sided ideal in \(R\) if and only if \(I\) is the kernel of some homomorphism \(f : R \to S\) for a suitable ring \(S\).

3.1.6 Theorem

A homomorphism \(f : R \to S\) is a monomorphism if and only if Ker(\(f\)) = (0).

  1. Suppose \(f : R \to S\) and \(g : S \to T\) are homomorphisms of rings. Then \(Ker(g \circ f) \supseteq Ker(f)\) (equality need not hold) . If \(g \circ f\) is a monomorphsim, then \(f\) is a monomorphism.
  2. Similarly, we have \(Im(g \circ f) \subseteq g(Im(f))\) (equality need not hold) . If \(g \circ f\) is an epimorphism, then \(g\) is an epimorphism.
  3. Any homomorphism \(f : R \to S\) from a simple ring \(R\) (to any ring S) is either identically zero or a monomorphism (because Ker(\(f\)) is a 2-sided ideal in \(R\))

3.2.1 The Epimorphism Theorem

Let \(f : R \to S\) be a homomorphism of rings with \(I = Ker(f)\). Then there exis ts a unique monomorphism \(T : R/I \to S\) such that the diagram commutes, i. e., \(f = \bar{f} \circ \eta\), where \(\eta\) is the natural map from \(R\) to \(R/I\).Moreover, \(T\) is an isomorphism if and only if \(f\) is an epimorphism.

3.2.2 Theorem: (Quotient of a quotient)

If \(I \subseteq J\) are both 2-sided ideals in \(R\), then \((R/I)/(J/I)\) is naturally isomorphic to \(R/J\).

3.2.3 Theorem

Suppose \(I\) and 1 are 2-sided ideals in a ring \(R\). Then the natural map \(\eta_{IJ} : R/I \to R/J\), defined by \(a + I \to a + J\) is well-defined if and only if \(I \subseteq J\) . Moreover, \(\eta_{IJ}\) is an epimorphism whose kernel is \(J/I\). Hence \((R/I)/(J/I)\) is ismomorphic to \(R/J\) .

3.2.5 Corollary

Let \(f : R \to S\) be a homomorphi8m of rings . Then we have the following.

  1. \(Ker(f)\) is a prime ideal if \(S\) is an integral domain.
  2. \(Ker(f)\) is a maximal ideal if \(f\) is surjective and \(S\) is a field.

3.2.6 Theorem

If \(I\) is a 2-sided ideal of \(R\), then \(I[X]\) is a 2-sided ideal of \(R[X]\) and the quotient ring \(R[X] /I[X]\) is naturally isomorphic to \((R/I)[X]\) .

Remarks:

  1. Image of an additive subgroup of \(R\) is an additive subgroup of \(S\), i. e., if \(A\) is a subgroup of \((R, +)\), then \(f(A)\) is a subgroup of \((S, +)\).
  2. Image of an ideal (left/right/2-sided) of \(R\) need not be an ideal in \(S\) unless\(f\) is an epimorphism.
  3. Inverse image of a subgroup of \((S, +)\) is a subgroup of \((R, +)\) and inverse image of an ideal in \(S\) is an ideal in \(R\).
  4. Inverse image of a prime ideal is a prime ideal in \(R\) if both \(R\) and \(S\) are commutative.
  5. Inverse image of a maximal ideal in \(S\) need not be a maximal ideal in \(R\) unless \(f\) is surjective.
  6. Homomorphic image of a PIR is a PIR

3.4.4 Theorem

For any commutative integral domain \(R\) with 1 , \(\mathbb{Q}(R)\) is the smallest field containing \(R\) as a subring, having the same identity as \(R\), smallest in the sense that if \(K\) is a field containing \(R\) as a subring, then \(K \supseteq \mathbb{Q}(R)\) as a subfield. This \(\mathbb{Q}(R)\) is unique upto isomorphism, called the field of fractions of R.

Chapter 4

4.1.4 Proposition

Let \(a, b \in R*\). Then \(a\) and \(b\) are associates of each other \(\iff a = ub\) for some unit \(u \in R \iff (a) = (b)\) .

4.1.8 Proposition

A prime is irreducible but not conversely.

4.1.9 Theorem

Let \(a\) be a non-zero non-unit in a commutativeintegral domain \(R\). Then we have the following.

  1. The element \(a\) is irreducible in \(R\) if and only if the ideal (a) is maximal among all principal ideals other than \(R\), i. e., there is no principal ideal in \(R\), other than \(R\), properly containing \((a)\).
  2. The element \(a\) is prime in \(R\) if and only if the ideal \((a)\) is a non-zero prime ideal in \(R\).

Remark

Given \(a, b \in R^*\), if gcd (resp. lcm) of \(a\) and \(b\) exists, then it is easy to see that it is unique upto associates and is denoted by gcd(\(a, b\)) (resp. lcm(\(a, b\))) .

4.2.2 Proposition

A Euclidean domain \(R\) has unity and whose group of units is given by \(U(R) =\) {\(a \in R^* | d(a) = d(1)\)}.

Examples of Euclidean domain

  1. The ring of integers \(\mathbb{Z}\) is Euclidean.
  2. The ring \(\mathbb{Z}[i]\) of Gaussian integers is Euclidean.
  3. \(R = K[X]\), the polynomial ring in one variable over a field \(K\) is Euclidean.
  4. \(R = K[[X]]\) , the formal power series ring in one variable over a field \(K\) is Euclidean.

4.3.1 Theorem

Every Euclidean domain is a PID ( with 1).

4.3.3 Theorem

Let \(R\) be a PID ( with 1 ) . Then we have the following.

  1. Every irreducible element is a prime in \(R\).
  2. Every non-zero prime ideal is maximal in \(R\).

4.3.4 Theorem

For a commutative integral domain R with unity, the following are equivalent.

  1. \(R\) is a field.
  2. R[X] is Euclidean.
  3. R[X] is a PID.

4.3.5 Theorem

Let \(R\) be a PID with 1 and \(a, b \in R^*\). Then we have

  1. \(d = gcd (a, b) \iff (d) = (a) + (b)\), (in particular, \(a, b\) are coprime to each other if and only if \(\exists x, y \in R\) such that \(1 = ax + by\)) and
  2. \(l = lcm (a, b) \iff (l) = (a) \cap (b)\).

4.4.2 Theorem

Every PID is a factorisation domain.

4.4.3 Proposition

If \(d\) is a positive integer, then the ring \(\mathbb{Z}[i\sqrt{d}]\) is a factorisation domain.

4.5.3 Theorem

A domain \(R\) is a UFD if and only if \(R\) is an FD in which every irreducible element is a prime

4.5.5 Corollary

Every PID is a UFD (by Theorems (4.3.3 ) ( 1 ) and (4.4.2) above)

4.5.10 Proposition

The content of a product of polynomial is the product of their contents and in particular, the product of primitive polynomials is primitive.

    NOTE:
  1. Any non-zero polynomial \(f(X)\) can be written as a product of a non-zero scalar and a primitive polynomial because \(f(X)/c(f)\) is primitive and \(c(f)\) is a non-zero scalar.
  2. Any irreducible polynomial is primitive.

4.5.11 Gauss Lemma

A primitive polynomial \(f(X) \in R[X]\) is irreducible in \(R[X]\) if and only if \(f(X)\) is irreducible in \(K[X]\) where \(K\) is the field of fraction of \(R\).

4.5.12 Theorem (Gauss)

Let \(R\) be a domain. Then \(R[X]\) is a UFD if and only if \(R\) is a UFD .
