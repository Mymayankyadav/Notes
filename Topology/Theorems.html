Theorem 17.1

Let \(X\) be a topological space. Then the following conditions hold:

  1. \(\phi\) and \(X\) are closed.
  2. Arbitrary intersections of closed sets are closed.
  3. Finite unions of closed sets are closed.

Theorem 17.2.

Let \(Y\) be a subspace of \(X\). Then a set \(A\) is closed in \(Y\) if and only if it equals the intersection of a closed set of \(X\) with \(Y\).

Theorem 17.4.

Let \(Y\) be a subspace of \(X\); let \(A\) be a subset of \(Y\) ; let \(\overline{A}\) denote the closure of \(A\) in \(X\). Then the closure of \(A\) in \(Y\) equals \(\overline{A} \cap Y\)

3.11 Theorem

The interior operation \(A \to A^{\circ}\) in a topological space \(X\) has the following properties

  1. \(A \subset A\).
  2. \((A^{\circ})^{\circ} =A^{\circ}\).
  3. \((A \cap B)^{\circ} =A^{\circ} \cap B^{\circ}\).
  4. \(X^{\circ} = X\).
  5. \(G\) is open iff \(G^{\circ} = G\).
Conversely, given any map \(A \to A^{\circ}\) of \(\mathbb{P}(X)\) into \(\mathbb{P}(X)\) in a set \(X\), satisfying 1 through 4, if open sets are defined in \(X\) using 5, the result is a topology on \(X\) in which the interior of a set \(A \subset X\) is just \(A^{\circ}\).

4.5 Theorem

Let \(X\) be a topological space and for each \(x \in X\), let ℬ\(_x\) be a nhood base at \(x\). Then

  1. if \(V \in\) ℬ\(_x\), then \(x\) \in \(V\),
  2. if \(V_1, V_2 \in\) ℬ\(_x\), then there is some \(V_3 \in\) ℬ\(_x\) such that \(V_3 \subset V_1 \cap V_2\),
  3. if \(V \in\) ℬ\(_x\), there is some \(V_0 \in\) ℬ\(_x\) such that if \(y \in V_0\), then there is some \(W \in\) ℬ\(_y\) with \(W \subset V\), and furthermore,
  4. \(G \subset X\) is open iff \(G\) contains a basic nhood of each of its points.
Conversely, in a set \(X\), if a collection ℬ\(_x\) of subsets of \(X\) is assigned to each \(x \in X\) so as to satisfy 1,2 and 3 and if we define “open” using 4, the result is a topology on \(X\) in which ℬ\(_x\) is a nhood base at \(x\), for each \(x \in X\).

4.7 Theorem

Let \(X\) be a topological space and suppose a nhood base has been fixed at each \(x \in X\). Then

  1. \(G \subset X\) is open iff \(G\) contains a basic nhood of each of its points,
  2. \(F \subset X\) is closed iff each point \(x \notin F\) has a basic nhood disjoint from \(F\),
  3. \(\overline{E}\) = {\(x \in X\) | each basic nhood of \(x\) meets \(E\)},
  4. \(E^{\circ}\) = {\(x \in X\) | some basic nhood of \(x\) is contained in \(E\)},
  5. Fr (\(E\)) = {\(x \in X\) | each basic nhood of \(x\) meets both \(E\) and \(X- E\)}.

4.8 Theorem

(Hausdorff criterion) For each \(x \in X\), let ℬ\(_x^1\) be a nhood base at \(x\) for a topology \(\tau_1\) on \(X\), and let ℬ\(_x^2\) be a nhood base at \(x\) for a topology \(\tau_2\) on \(X\). Then \(\tau_1 ⊂ \tau_2\) iff at each \(x \in X\), given \(B^1 \in\) ℬ\(_x^1\) , there is some \(B^2 \in\) ℬ\(_x^2\) such that \(B^2 ⊂ B^1\)

6.3 Theorem

If \(A\) is a subspace of a topological space \(X\), then:

  1. \(H\) \(\subset\) \(A\) is open in \(A\) iff \(H\) = \(G\) \(\cup\) \(A\), where \(G\) is open in \(X\),
  2. \(F\) \(\subset\) \(A\) is closed in \(A\) iff \(F = K\) \(\cup\) \(A\) where \(K\) is closed in \(X\),
  3. if \(E\) \(\subset\) \(A\), then \(Cl_A E\) = \(A\) \(\cup\) \(Cl_X E\),
  4. if \(x\) \(\in\) \(A\), then V is a nhood of x in \(A\) iff V = U \(\cup\) \(A\), where U is a nhood of \(x\) in \(X\),
  5. if \(x\) \(\in\) \(A\), and if ℬ\(_x\) is a nhood base at \(x\) in \(X\), then {\(B\) \(\cup\) \(A\) | \(B\) \(\in\) ℬ\(_x\)} is a nhood base at \(x\) in \(A\),
  6. if ℬ is a base for \(X\), then {\(B\) \(\cup\) \(A\) | \(B\) \(\in\) ℬ} is a base for \(A\).

Theorem 18.1

Let \(X\) and \(Y\) be topological spaces; let \(f : X \to Y\). Then the following are equivalent:

  1. \(f\) is continuous.
  2. For every subset \(A\) of \(X\), one has \(f (\overline{A}) \subset \overline{f (A)}\).
  3. For every closed set \(B\) of \(Y\), the set \(f^{-1}(B)\) is closed in \(X\).
  4. For each \(x \in X\) and each neighborhood \(V\) of f (x), there is a neighborhood \(U\) of \(x\) such that \(f (U) \subset V\) .
If the condition in 4 holds for the point \(x\) of \(X\), we say that \( f\) is continuous at the point \(x\).

7.3 Theorem

If \(X\), \(Y\) and \(Z\) are topological spaces and \(f : X \to Y\) and \(g : Y \to Z\) are continuous, then \(g \circ f : X \to Z\) is continuous.

7.5 Theorem

Restriction, on the subset, of continuous function is continuous.
